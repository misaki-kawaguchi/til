# シングルファイルコンポーネントをローカル登録する

## LikeHeader.vue
```
<template>
  <h1>いいね</h1>
</template>
```

## App.vue
```
<template>
  <div>
    <LikeHeader></LikeHeader>
  </div>
</template>

import LikeHeader from "./components/LikeHeader.vue";

<script>
export default {
  components: {
    LikeHeader
  }
}
</script>
```

### コンポーネントの命名規則
・ケバブケース（<like-header>）かパスカルケース（LikeHeader）にする
