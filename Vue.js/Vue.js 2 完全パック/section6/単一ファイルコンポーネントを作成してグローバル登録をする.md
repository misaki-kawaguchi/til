# 単一ファイルコンポーネントを作成してグローバル登録をする

## LikeNumber.vue
```
<template>
  <p>いいね({{ number }})</p>
</template>

<script>
export default {
  data: function() {
    return {
      number: 5
    }
  }
}
</script>
```

## App.vue
```
<template>
  <LikeNumber></LikeNumber>
</template>
```

## main.js
```
import LikeNumber from "./LikeNumber.vue"

VUe.component('LikeNumber', LikeNumber);
```
