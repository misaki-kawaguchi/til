# 2.CSSアニメーションを使ってslideするトランジションを作成

## App.vue
```
<template>
  <div class="main">
    <button @click="show = !show">切り替え</button>

    <transition name="slide">
      <p v-if="show">bye</p>
    </transition>

    // 単一の要素しか利用できない
    <transition name="slide">
      <div v-if="show">
        <p>bye</p>
        <p>hello</p>
        <p>hello</p>
        <p>hello</p>
      </div>
    </transition>
  </div>
</template>

<script>
export default {
  data() {
    show: true
  }
}
</script>

<style scoped>
  // 現れる時のトランジションの状態
  .slide-enter-active {
    animation: slide-in 0.5s;
  }

  // 消える時のトランジションの状態
  .slide-leave-active {
    animation: slide-in 0.5s reverse;
  }

  @keyframes slide-in {
    from {
      transform: translateX(100px)
    }
    to {
      transform: translateX(0)
    }
  }

  .main {
    width: 70%;
    margin: auto;
    padding-top: 5rem;
    text-align: center;
  }
</style>
```
