# 06.name属性を動的に変更して、動的トランジションを作成する
## App.vue
```
<template>
  <div class="main">
    <button @click="myAnimation = 'slide'">Slide</button>
    <button @click="myAnimation = 'fade'">Fade</button>

    <button @click="show = !show">切り替え</button>

    <transition
      :name="myAnimation"
      appear
    >
      <p v-if="show">bye</p>
    </transition>

  </div>
</template>

<script>
export default {
  data() {
    show: true,
    myAnimation: 'slide'
  }
}
</script>

<style scoped>

  // 現れる時の最初の状態
  .fade-enter {
    opacity: 0;
  }

  // 現れる時のトランジションの状態
  .fade-enter-active {
    transition: opacity 2s;
  }

  // 現れる時の最後の状態
  .fade-enter-to {
    opacity: 1;
  }

  // 消える時の最初の状態
  .fade-leave {
    opacity: 1;
  }

  // 消える時のトランジションの状態
  .fade-leave-active {
    transition: opacity 2s;
  }

  // 消える時の最後の状態
  .fade-leave-to {
    opacity: 0;
  }


  // 現れる時の最初の状態
  // 消える時の最後の状態
  .slide-enter,
  .slide-leave-to {
    opacity: 0;
  }

  // 現れる時のトランジションの状態
  .slide-enter-active {
    animation: slide-in 0.5s;
    transition: opacity 1s;
  }

  // 消える時のトランジションの状態
  .slide-leave-active {
    animation: slide-in 0.5s reverse;
    transition: opacity 1s;
  }

  @keyframes slide-in {
    from {
      transform: translateX(100px)
    }
    to {
      transform: translateX(0)
    }
  }

  .main {
    width: 70%;
    margin: auto;
    padding-top: 5rem;
    text-align: center;
  }
</style>
```
