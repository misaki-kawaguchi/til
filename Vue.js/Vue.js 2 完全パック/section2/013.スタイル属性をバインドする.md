# 013スタイル属性を、オブジェクトを用いて動的にバインドする

## HTML
```
<div id="app">
   <h1 :style="{color: textColor, 'background-color': bgColor}">Hello</h1>
</div>
```

## JavaScript
```
new Vue({
  el: '#app',
  data: {
    textColor: 'red',
    bgColor: 'blue'
  }
})
```

## スタイルオブジェクトを使った場合

## HTML
```
<div id="app">
   <h1 :style="styleObject">Hello</h1>
</div>
```

## JavaScript
```
new Vue({
  el: '#app',
  data: {
    styleObject: {
      color: 'red', 
      'background-color': 'blue'
    }
  }
})
```

## 複数のスタイルを指定する場合
## HTML
```
<div id="app">
   <h1 :style="[styleObject, baseStyles]">Hello</h1>
</div>
```

## JavaScript
```
new Vue({
  el: '#app',
  data: {
    styleObject: {
    color: 'red', 
    'background-color': 'blue'
  },
    baseStyles: {
      fontSize: '60px'
    }
  }
})
```
