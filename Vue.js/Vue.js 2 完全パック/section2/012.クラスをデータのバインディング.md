# 012.クラスをデータにバインディングする

## HTML
```
<div id="app">
   <h1 :class="classObject">Hello</h1>
   <button @click="isActive = !isActive">切り替え</button>
</div>
```

## css
```
.red {
  color: red;
}

.bg-blue {
  background-color: blue;
}
```

## JavaScript
```
new Vue({
  el: '#app',
  data: {
    isActive: true
  },
  computed: {
    classObject: function() {
    return { 
      red: this.isActive, 
      'bg-blue': !this.isActive 
      }
    }
  }
})
```

ケバブケースの場合は`'bg-blue'`と書く

## 配列を使った場合
## HTML
```
<div id="app">
  <h1 :class="[color, bg]">Hello</h1>

  <h1 :class="[{ red: isActive}, bg]">Hello</h1>
</div>
```

## JavaScript
```
new Vue({
  el: '#app',
  data: {
    isActive: true,
    color: 'red',
    bg: 'bg-blue'
  },
  computed: {
    classObject: function() {
      return { 
        red: this.isActive, 
        'bg-blue': !this.isActive 
      }
    }
  }
})
```
