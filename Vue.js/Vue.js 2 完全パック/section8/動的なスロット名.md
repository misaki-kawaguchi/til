# 動的なスロット名

## App.vue(親)
```
<template>
  <div>
    <LikeHeader>
      <template v-slot:[title]></template>
    </LikeHeader>
  </div>
</template>

<script>
import LikeHeader from "./components/LikeHeader.vue";

export default {
  data() {
    return {
      number: 10,
      title: "title"
    };
  },
  components: {
    LikeHeader
  }
};
</script>
```

`v-slot:[title]`は`#[title]`に置き換えることができる

<LikeHeader></LikeHeader>などに使う場合は、`<LikeHeader #default="slotProps">`というようにdefaultを追加してあげる
