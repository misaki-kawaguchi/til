# Androidのデータ保存について

|  方法  |  内容  |
| :---- | :---- |
|  SQLiteデータベース  |  Android OSにあらかじめ備わっているSQLiteデータベースをにデータを保存する  |
|  プレファンス  |  簡易データをキーと値のペアで保存する。アプリの設定値の保存に向いている  |
|  内部ストレージ  |  端末内部にファイル形式で保存する  |
|  外部ストレージ  |  SDカードなどにファイル形式で保存する。最近は端末はカードスロットそのものがなく、内部ストレージで代用しているものが多い  |
|  クラウド  |  ネットワーク経由でサーバーにデータを保存する。通常のWebアプリと同様にHTTPプロトコルを使用する  |
