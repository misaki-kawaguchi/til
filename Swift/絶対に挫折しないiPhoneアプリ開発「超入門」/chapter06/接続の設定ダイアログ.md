# 接続の設定ダイアログ

|  種類  |  説明  |
| ---- | ---- |
|  Outlet  |  ストーリーボード上のUI部品をプロパティとして接続する。UI部品の見た目をコードで変更したい場合に指定する。  |
|  Action  |  ストーリーボード上のUI部品をメソッドとして接続する。「UI部品がタップされた」のように、UI部品に対して何らかのアクションが発生した際に特定の処理を実行する場合に指定する。  |


|  項目  |  説明  |
| ---- | ---- |
|  Connection  |  接続の種類（Outlet or Action）  |
|  Object  |  ストーリーボードと接続するクラス（接続先のクラスの名前が自動的に表示される）  |
|  Name  |  接続するUI部品の名前を設定する。Outlet接続の場合はプロパティ名、Sction接続の場合はメソッド名となる。  |
|  Type  |  接続するUI部品の型を設定する（通常はクラス名）  |
|  Storage  |  Outlet接続の場合のみ表示される。プロパティのメモリ管理に関する設定を行う  |
|  Event  | Action接続の場合のみ表示される。イベント（ユーザーによるiPhoneの操作の内容）を設定する。   |
|  Arguments  |  Action接続の場合のみ表示される。引数を設定する。None：メソッドに引数を渡さない、 Sender：メソッドに接続する部品を引数として渡す、Sender and Event :メソッドに接続する部品をイベントを引数として渡す |
